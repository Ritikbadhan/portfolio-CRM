# Development Guide

## Prerequisites
- Node.js >= 20
- MongoDB Local/Cloud instance
- Git

## Setup
1. Clone the repository
2. Run `npm install`
3. Setup `.env` files.

## Workflow
- Create a feature branch.
- Write tests.
- Run `npm run lint` and `npm run format`.
- Commit changes (Husky will ensure format & lint on pre-commit).
